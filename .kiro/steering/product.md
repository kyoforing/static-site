# Product Overview

Interactive learning web page for Chapter 7 ("需要學習的技能" / Skills to Learn) of a Vibe Coding book. Written in Traditional Chinese (zh-TW).

The page presents educational content about core developer skills in the AI era — feedback loops, modular systems, embracing learning, and honing craft — followed by a 5-question multiple-choice quiz that tests comprehension.

Key user flows:
- Read chapter content across 5 sections with sticky nav and scroll-based highlighting
- Take a quiz: select answers → submit → view score + per-question feedback → reset and retry
- Score is calculated as `correctCount × 20` (range 0–100), with tiered feedback messages
